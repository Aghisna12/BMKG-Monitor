# 🌦️ BMKG Monitor

Website monitoring cuaca berbasis data dari BMKG (Badan Meteorologi, Klimatologi, dan Geofisika) Indonesia.
Menampilkan informasi cuaca secara sederhana, cepat, dan ringan untuk diakses dari berbagai perangkat.

🔗 **Live Preview:**
👉 [https://bmkg-monitor.web.app](https://bmkg-monitor.web.app)

---

## ✨ Fitur Utama

* 🌍 Data cuaca dari BMKG (real-time / near real-time)
* ⚡ Performa cepat (static & ringan)
* 📱 Responsive (mobile, tablet, desktop)
* 🌙 UI sederhana & nyaman dilihat
* 🔍 Mudah digunakan tanpa login

---

## 🛠️ Teknologi yang Digunakan

* HTML, CSS, JavaScript
* Fetch API (ambil data BMKG)
* Hosting: Firebase Hosting
* Static Web App (tanpa backend berat)

## 🚀 Cara Menjalankan Secara Lokal

1. Clone repository:

```bash
git clone https://github.com/Aghisna12/BMKG-Monitor.git
```

2. Masuk ke folder:

```bash
cd BMKG-Monitor
```

3. Jalankan dengan Live Server / buka langsung:

```bash
index.html
```

---

## 🌐 Deployment

Project ini di-deploy menggunakan **Firebase Hosting**.

Jika ingin deploy sendiri:

```bash
npm install -g firebase-tools
firebase login
firebase init
firebase deploy
```
